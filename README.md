This package is one of the components used for `volcano` â€” the output of Group 4 for the ChannelFix Hackathon

## Install
```
$ npm install volcano-cli
```

## Usage
```
$ volcano serve
$ volcano help
$ volcano version
```

## Options
`serve`       - provides a random port to host

`help`        - dislplays how the package can be used

`version`     - displays the current version of the installed package
